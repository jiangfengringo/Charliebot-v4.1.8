Using Anna

The first thing that should be done is to run the interpreter and experiment with Anna. Execute run.bat in Windows or server.sh in Linux. Anna will say her response to "Hello". Type in a phrase and hit enter. You will see which <pattern> tag was matched, and the response will be displayed on your screen. Now you're talking to Anna!

You are asked to help improve Anna by having conversations and then editing the files to ensure better responses.

After one patch and a SourceForge username (free reg), you will be given developer access to the project so you can take a more active role.
