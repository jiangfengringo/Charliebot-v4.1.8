Downloading Anna

Please note, in order to run the interpreter you will need Java 2 1.4, available from java.sun.com

Anna is released in 2 "sizes" of 2 flavors. The bundle should be downloaded if you are just getting started on the project, or if there has been a change in configuration as noted in the ChangeLog. The AIML only release should be downloaded in all other cases.

Download the .zip version for Windows or Mac, or the .tar.gz version for Linux.
